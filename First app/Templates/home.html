Meetup Summaries

Meetup #1

We decided to make our own CodeCatz website. For this purpose we made our own repository on Github called litterbox and learned how to send pull requests and use push to share the cool commits we've made with others.

Meetup #2

We decided to make our very first app The Complexity of Life Calculator. We learned how to use variables and if/else statements in python.

Meetup #3

We started adding different functions to our Complexity of Life Calculator. We also found Flask which is a lightweight web application framework written in Python. Tutorial: How to install Flask.

Meetup #4

We used Flask to make a simple web app. It became prettier with the help of Bootstrap.

Meetup #5

We reviewed how to make a simple app using Flask, used Bootstrap again to make it look more fancy, practiced different CSS tricks and successfully debugged all the problems we've encountered along the way.

Meetup #6

We spent most of the time reviewing the code for our complexicators and the way they work with the Flask web app. We also talked about HTML form validation and forking.

Meetup #7

We learned about jQuery, a fast, small, and feature-rich JavaScript library. We also found jsFiddle which is a great tool to experiment with JavaScript and various js libraries, including jQuery.

Meetup #8

We spent most of the time playing with jQuery again. We also had a very cool presentation about User Experience by one of our kittens.

Meetup #9

This week we were hosted by Celtra. Most of the mentors were absent, so we used most of the time to cath up and review github fetch command. One of the members also had a wonderful presentation about Celtra's rich media mobile advertising.

Meetup #10

We talked about Raspberry Pi and DNS Servers.

Meetup #11

We were hosted by Celtra again where we were working on our homework - making our own websites.
